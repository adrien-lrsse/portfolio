## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/adrien-lrsse/portfolio.git
    ```
2. Install dependencies:
    ```sh
    npm install
    ```
3. Start the development server:
    ```sh
    npm run dev
    ```
